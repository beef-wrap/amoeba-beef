# About

A Beef wrapper for [Amoeba](https://github.com/starwing/amoeba), a Cassowary constraint solving algorithm implemented in pure C.

# Building static libs

```sh
zig build --prefix libs
```

# Running the example

Clone the repo, and open up the workspace in the Beef IDE, and run.
